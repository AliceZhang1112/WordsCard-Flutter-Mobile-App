# 📖 Wordscard - Flutter Mobile App

Welcome to **Wordscard**, a simple yet elegant mobile application built using **Flutter**! This project was designed for anyone looking to expand their vocabulary, especially those learning a new language. 🚀

## 🎯 Features
✅ Displays a list of word pair suggestions 📚  
✅ Allows users to favorite word pairs 💖  
✅ Smooth UI built with Flutter Widgets 🏗️  
✅ State management with `StatefulWidget` 🛠️  
✅ Cross-platform compatibility (Android & iOS) 📱  
✅ Mobile widget support to display random words on the home screen, helping users memorize words in short bursts of time 📝

## 🏗️ Tech Stack
- **Flutter**: Dart-powered UI framework 🌐  
- **Dart**: Modern programming language optimized for UI development 🚀  
- **Provider (Optional)**: Simple state management solution ⚙️

## 💡 Why This Project?
This project demonstrates my hands-on experience with Flutter development, state management, and UI design. It serves as an example of my ability to build and structure scalable mobile applications efficiently. Additionally, the widget feature makes it easy for users to memorize words passively in their spare time.
